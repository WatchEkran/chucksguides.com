6.1 – WAYPOINT NAVIGATION
D: CONSULTING STEERPOINT DATA FROM CDU
1. Set Auxiliary Avionics Panel (AAP) Steerpoint
Selector Knob to MISSION (you access all
waypoints available for the current mission).
2. To consult data on the selected Steerpoint,
set Auxiliary Avionics Panel (AAP) CDU
(Control Display Unit) Selector Knob to
STEER. This will display the STEERPOINT
INFO Page.
